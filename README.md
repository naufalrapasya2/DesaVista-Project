## 📋 Deskripsi Proyek
DesaVista adalah aplikasi web untuk promosi dan informasi Desa Wisata berbasis digital di Yogyakarta. Aplikasi ini mendukung konsep:
- ✅ Digitalisasi layanan desa
- ✅ Promosi wisata
- ✅ Kemudahan akses informasi

## 🎯 Fitur Utama
1. **Informasi Destinasi Wisata Lengkap** (16+ destinasi)
   - Candi Borobudur, Prambanan, Malioboro, Tamansari
   - Pantai Parangtritis, Drini, Junew Blue Ocean
   - Goa Pindul, Air Terjun Sri Getuk
   - HeHa Sky View, Bukit Panguk Kediwung
   - Gunung Api Purba Nglanggeran
   - The Lost World Castle, Gembira Loka Zoo
   - Museum Benteng Vredeburg, Jeep Lava Tour Merapi
   
2. **Layanan Publik Terintegrasi**
   - 🏥 Rumah Sakit (4+ RS dengan CP & alamat lengkap)
   - 🏢 Kantor Pelayanan (Dukcapil, Kelurahan, Kecamatan)
   - 🏛️ Pemerintah Daerah (Dinas Pariwisata, Kebudayaan, BPBD)
   - 🎓 Lembaga Pendidikan (SD, SMP, SMA, Universitas)

3. **Detail Informasi Interaktif**
   - Modal popup untuk detail destinasi
   - Tab navigation untuk kategori informasi
   - Filter dan pencarian

## 🛠️ Teknologi
- HTML5 (Semantic)
- CSS3 (Flexbox, Grid, Variables, Animations)
- Vanilla JavaScript (ES6+)
- JSON (Data Storage)
- GitHub (Version Control)
